# DigitalPro

A single-page, Tailwind CSS template for a marketing agency. Developed as part of the **SEF-Academy Front-End Course**.

🔗 **Live Demo:** [devfadyhany.github.io/DigitalPro](https://devfadyhany.github.io/DigitalPro/)

---

## 📋 Overview

DigitalPro is a front-end marketing agency website, built using HTML, Tailwind CSS.

## 🛠️ Built With

- **HTML5** — Semantic markup across multiple pages
- **Tailwind CSS** — CSS utility classes for responsive styling
- **Font Awesome** — Icons via webfonts
